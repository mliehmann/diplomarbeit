Game controls:

'ESC':			exit game
'F10'			start game / next level
'F12'			end level (you loose)
'F1'			pause game
'print'			save screenshot

Default key bindings:

keypad '-' und '+':	change mouse sensitivity	(SDLKey 269 and 270)
'M'			pause music			(SDLKey 109)

mouse motion:		gear / pitch
left mouse button	shoot
right mouse button	thruster
middle mouse button	stop ship rotation

'Y'			shoot				(SDLKey 122)
'X' and 'C':		roll				(SDLKey 120 and 99)
'SPACE':		thruster			(SDLKey 32)
'V'			stop ship rotation		(SDLKey 118)

To change key bindings check 'typedef_sdlkey.txt' and set appropriate values within 'arp.cfg'.
Resolution, sound, crosshair etc. can also be configured in 'arp.cfg'.
Path names must not contain white spaces!
